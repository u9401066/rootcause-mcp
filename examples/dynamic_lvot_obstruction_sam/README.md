## ðŸ”‘ Key Teaching Points

### 1. Pre-op Red Flags (Missed)
- **Systolic murmur at LSB** - Could be HOCM, not just "flow murmur"
- **"Mild AR"** - Does NOT rule out HOCM (can coexist)
- **3-year-old echo** - Needs repeat in elderly with murmur
- **Hypertension** - LVH is substrate for SAM

### 2. Intra-op Red Flags (Missed)
- **A-line Bisferiens/Spike-Dome** - PATHOGNOMONIC for HOCM
- **No response to Ephedrine** - Should trigger differential
- **Worsening with Epinephrine** - THE diagnostic clue
- **Hyperdynamic LV on TEE** - Not hypovolemia if "kissing walls"

### 3. The Rule
> **"If shock gets WORSE with Epinephrine, think Dynamic Obstruction"**

### 4. Memory Aid: Treatment is OPPOSITE
- Normal shock: Push fluids, give inotropes
- SAM: Push fluids, **STOP** inotropes, give **pure Î±-agonist**
